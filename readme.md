## Table of Contents

### Installation

1. Clone repository
```
$ git clone https://github.com/Tbs-Rachit/movie-search-nodejs.git
```

2. Enter folder
```
$ cd movie-search-nodejs
```

3. Install npm dependencies
```
$ npm install
```

### Run

1. Run the project
```
$ node app.js
```
